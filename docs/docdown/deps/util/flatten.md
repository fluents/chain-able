(Function): flatten multi-dimensional arrays in `1` line

#### Arguments
1. `arr=undefined` *(Array|any)&#91;&#93;): array(s)* to flatten

#### Returns
*(&#42;)*: flattened arrays

#### Example
```js
flatten([[1], [2]])
//=> [1, 2]
flatten([[1], 2])
//=> [1, 2]
flatten(1)
//=> [1]

```
