#### Arguments
1. `value=undefined` *(&#42;)*: value in array iteration
2. `index=undefined` *(number)*: current index
3. `arr=undefined` *(Array)*: array being iterated, `thisArg` when using .filter

#### Returns
*(Array)*: arr

#### Example
```js
var list = [1, 2, 3, 1, 2, 3, 1, 2, 3]

list.filter(uniq)
//=> [1, 2, 3]

```
